# Contributing

This is primarily a personal portfolio and project-index repository.

## Suggestions

If you notice a broken link, typo, unclear project description, or missing project status, open an issue or pull request.

## Project Code

Most actual project code lives in separate repositories linked from this index. For code changes, open the issue or pull request in the specific project repository instead of this portfolio repository.

## Style

- Keep project descriptions concise.
- Prefer links to real repositories or deployed demos when available.
- Keep status labels honest: planned, restart planned, in progress, completed MVP, completed, archived.
