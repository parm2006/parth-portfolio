# Terraria Pixel Art Converter

**Status:** Completed CLI, frontend planned  
**Repo:** [parm2006/terraria-pixel-converter](https://github.com/parm2006/terraria-pixel-converter)

Tool that converts pixel art images into Terraria block/wall material lists.

## Current Direction

- Existing CLI converts image colors into Terraria block/wall material counts.
- Planned web interface would make the tool accessible without local Python setup.
- Useful for Terraria builders who want a faster way to plan pixel art.

## Frontend Workflow

- Upload image.
- Choose blocks or walls.
- Adjust size and palette options.
- Preview output.
- Get a materials list.
