# Scholarly Search Engine

**Status:** In progress  
**Repo:** [parm2006/Scholarly-Search-Engine](https://github.com/parm2006/Scholarly-Search-Engine)

Fairness-aware academic search engine for finding research papers from a user query, with emphasis on surfacing contributions from women and people of color.

## Current Direction

- Keep improving the existing backend/frontend rather than treating this as not started.
- Use the code analysis document in the local project as a roadmap.
- Prioritize correctness and basic usefulness before large-scale semantic infrastructure.
- Eventually, this can become the search/retrieval foundation for Particle.

## Likely Next Work

- Fix basic backend/API issues.
- Improve search ranking and pagination.
- Rebuild or improve frontend search experience.
- Add fairness-aware ranking transparently.
- Later: semantic search, citation graph, embeddings, and Particle integration.
