# Security Policy

This repository is an index of projects and does not contain production application code.

## Reporting Security Issues

If you find a security issue in one of the linked projects, report it in that project's repository when possible.

If the issue is about this portfolio repository itself, open a private report through GitHub security advisories if available, or contact the repository owner directly.

Please do not publicly disclose sensitive credentials, tokens, private URLs, or exploitable vulnerabilities.
